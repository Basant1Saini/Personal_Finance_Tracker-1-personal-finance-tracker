# Personal Finance Tracker

A simple web application to track personal income, expenses, and budgets.

## Development Plan for Beginners

### Phase 1: Project Setup (Week 1)
- [ ] Set up development environment
- [ ] Create basic HTML structure
- [ ] Add CSS for styling
- [ ] Set up JavaScript files
- [ ] Create project folder structure

### Phase 2: Core Features (Week 2-3)
- [ ] Add income tracking
- [ ] Add expense tracking
- [ ] Create category system
- [ ] Build basic calculator
- [ ] Add data storage (localStorage)

### Phase 3: User Interface (Week 4)
- [ ] Create dashboard view
- [ ] Add forms for input
- [ ] Build transaction list
- [ ] Add edit/delete functions
- [ ] Make responsive design

### Phase 4: Advanced Features (Week 5-6)
- [ ] Add budget planning
- [ ] Create simple charts
- [ ] Add date filtering
- [ ] Export data feature
- [ ] Add data validation

### Phase 5: Polish & Deploy (Week 7)
- [ ] Test all features
- [ ] Fix bugs
- [ ] Improve UI/UX
- [ ] Add documentation
- [ ] Deploy to GitHub Pages

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Start coding!
